# Page Weaver Studio

## Project info

A powerful, algorithmic page builder for high-performance teams.

## Getting Started

Follow these steps to run the project locally:

1. **Install dependencies**
   ```sh
   npm install
   ```

2. **Start the development server**
   ```sh
   npm run dev
   ```

## Technologies

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Firebase (Auth & Firestore)

## Features

- Real-time page builder
- Arabic and English language support
- Analytics dashboard
- Messaging system
- Team management
